## Run Locally

Clone the project

```bash
  git clone https://github.com/Arkar20/next-refresh
```

Go to the project directory

```bash
  cd next-refresh
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Nextjs, Shadcn, Jest, Cypress, Prisma
